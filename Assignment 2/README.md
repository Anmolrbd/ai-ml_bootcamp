# Python OOP Assignment

This repository contains the Python programs I wrote while learning Object-Oriented Programming (OOP). The programs are based on OOP concepts and examples that I studied and practiced.

## Files

### 01_student_management.py

A simple program that demonstrates:

* Classes and objects
* Constructors
* Instance methods
* Updating object information

### 02_inheritance.py

A program showing how inheritance works using `Person`, `Student`, and `Teacher` classes.

### 03_polymorphism.py

A program demonstrating method overriding and polymorphism using different animal classes.

### 04_bank_account.py

A simple bank account program that demonstrates encapsulation using a private balance attribute along with deposit and withdrawal methods.

## Language Used

* Python 3

These programs were created as part of my OOP assignment to practice the basic concepts of object-oriented programming in Python.
